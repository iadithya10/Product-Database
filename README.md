 Product API (Node.js + Express + MySQL)

This project is a simple CRUD REST API for managing products using Node.js, Express, and MySQL.
It demonstrates server setup, database connection, table creation, and API implementation using best practices.

 Features:

Express server setup

MySQL database connection

Products table 

REST APIs:

Create product (POST)

Get all products (GET)

Update product (PUT)

Delete product (DELETE)

Pagination implemented in the Get Products API using page and limit query parameters to efficiently handle large datasets
